CS 3113

  • Course name: Principles of Cyber Security
  • Course number: CS 3113
  • Description: An introductory course in Cyber Security including an examination of the fundamental principles underlying cyber security, how these principles interrelate and how they are typically employed to secure computer systems and networks.
